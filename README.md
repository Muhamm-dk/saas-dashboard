# AWS SaaS Sales Analytics Dashboard

AI-powered lead scoring and customer segmentation system for B2B SaaS companies.

## 🎯 Features

- **Lead Scoring**: Predict high-value deals (0-100 score)
- **Customer Segmentation**: Identify 5 customer personas
- **Batch Processing**: Score hundreds of leads at once
- **Interactive Dashboard**: Real-time predictions with visualizations

## 📊 Model Performance

- Lead Scoring ROC-AUC: 0.85+
- Customer Segments: 5 validated personas
- Training Data: 9,994 B2B SaaS transactions
